## Scripts

```bash
pnpm dev          # Serveur de développement
pnpm build        # Build production
pnpm start        # Serveur production
pnpm lint         # Linter ESLint
pnpm format       # Formatage Prettier
pnpm type-check   # Vérification TypeScript
```
